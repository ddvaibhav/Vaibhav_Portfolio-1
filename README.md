## ⚙️ Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/ddvaibhav/Vaibhav_Portfolio-1.git
cd Vaibhav_Portfolio-1

# 2. Install dependencies
npm install

# 3. Add environment variable
echo "RESEND_API_KEY=your_key_here" > .env

# 4. Start the server
node server.js
```

Then open 👉 `http://localhost:10000`
